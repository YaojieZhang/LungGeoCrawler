# GEO Single-cell Research Crawler (Beta 1.0)
本项目是一个专为肺癌单细胞研究（或扩展至其他生物医学领域）设计的元数据抓取与深度分析工具。它通过结合 NCBI Entrez API、静态网页爬虫以及大语言模型（LLM），能够自动化地从 GEO 数据库中提取、处理并分析复杂的数据集信息。
